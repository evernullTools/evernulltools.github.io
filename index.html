Current Product

SlashForge — Combat Trail Creator

SlashForge is a Unity tool for creating customizable combat trails for weapons, melee attacks, and stylized action effects.

It is designed for fast setup, editable presets, animation-event workflows, and deep visual control, making it easier to create sword trails, elemental slashes, energy arcs, stylized swipe kicks, and other combat motion effects directly inside Unity.

What It Does

Built for fast setup and deep control.

SlashForge is designed to help users start quickly with presets, while still giving technical artists room to customize the result.

Preset-based workflow

Start from ready-made trail styles and edit the look without rebuilding the effect from zero.

Runtime trail mesh

Generates lightweight procedural trail geometry between weapon tip and base transforms.

Shader-friendly data

Passes useful lifetime, fade, stretched UV, and distance-based UV data into the generated mesh.

Animation events

Control when trails start, end, and fade out using animation events or code.

Custom material support

Use the built-in TrailProfile workflow or bypass it with your own shader/material setup.

Documentation-first

Documentation and showcase videos are hosted externally to keep the Unity package lightweight and easy to download.
