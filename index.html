Current Product

SlashForge - Combat Trail Creator

A Unity tool for creating customizable combat trails for weapons, melee attacks, and stylized action effects.

Fast setup

Add the component, assign tip and base transforms, choose a TrailProfile, and start controlling trails through animation events or code. Make trails in minutes!

Editable presets

Start from ready-made trail styles, then customize the visuals easily with profile controls, gradients, textures, dissolve settings, and shader options.

Runtime mesh generation

SlashForge generates lightweight procedural mesh geometry between weapon points, allowing the trail shape to follow animation motion directly.

Custom shader support

Use the built-in TrailProfile workflow or bypass it with a custom material while still receiving useful mesh data like vertex colors and UVs that help you create the effect you want.
